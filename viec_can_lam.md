# DANH SÁCH VIỆC CẦN LÀM (hàng đợi tự động - agent cloud xử lý mỗi 5 tiếng)

Cách dùng: thêm việc vào mục **Việc đang chờ** dưới dạng `- [ ] nội dung việc`
(sửa file này trực tiếp trên GitHub web cũng được - bấm biểu tượng bút chì).

Agent cloud mỗi 5 tiếng sẽ đọc file này:
- Việc làm được trên cloud (sửa README/tài liệu trong repo, kiểm tra lượt tải,
  soạn nội dung, phân tích...) → làm xong chuyển xuống **Việc đã xong** với `- [x]` kèm kết quả.
- Việc cần laptop của chủ dự án (sửa code app, build exe, đăng bản mới) → agent sẽ
  ghi chú "⏳ cần làm trên laptop" và giữ nguyên; những việc đó nhắn trực tiếp
  Claude Code trên laptop.
